Rounded Tables

Example 1: Create Rounded Corners Using Images (a HTML trick)
  1. Switch to Code or Design View.
  2. Create a table of 3 x3. Set border (by default), cellspacing and cellpadding to ZERO (0).
  3. Drag and drop respective image to a data-cell at each corner.
  4. Go to Row 1, Col 2, set the width of the data-cell to 100%.
  5. Set the table's background same as the image colour.
  6. Place content in the centre.
